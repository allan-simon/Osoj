Osoj by sysko

Open source online judge

This project aimed to create an open source plateform to set up an online judge (a website which contains programming exercise that can be submitted online and corrected automatically) For the moment it only support C and C++

Dependencies

Python 2.6 Webpy Gcc

Install

Create a clone of the repository, Compile the library in the libc folder ( cd libc && make ) (optional), delete the openoj.db and create a fresh one by doing sqlite3 openoj.db < ressources/sql/initdb.sql To run the server python server.py [port] (by default it's listening on 8080) If you want to add some exercise go on the URL /admin/add-exercise
